API ENDPOINTS SUMMARY
================================================================================

1. GET /api/routes/comprehensive
   Purpose: Get multi-source route data
   Data Sources: Google Maps, OSM, OpenAI, Grok
   Returns: Routes with traffic, AI analysis, impact assessment

2. GET /api/roads/comprehensive
   Purpose: Get road network data for a location
   Data Sources: Google Maps, OSM, OpenAI
   Returns: Roads with properties, condition, traffic

3. GET /api/routes/trace
   Purpose: Trace route using addresses (with geocoding)
   Data Sources: Google Maps, OpenAI
   Returns: Complete route from address names

4. GET /api/roads/detailed
   Purpose: Get detailed road information with filters
   Data Sources: OSM, OpenAI, Grok
   Returns: Roads filtered by type with full analysis

5. GET /api/routes/traffic-analysis
   Purpose: Analyze traffic patterns and predictions
   Data Sources: Google Maps, Historical data
   Returns: Traffic conditions with ETA estimates

6. GET /api/routes/optimization
   Purpose: Optimize multi-stop routes
   Data Sources: Google Maps
   Returns: Optimized waypoint order and segments

7. POST /api/routes/safety-assessment
   Purpose: Assess safety of a route
   Data Sources: All sources
   Returns: Safety score, risk level, recommendations
